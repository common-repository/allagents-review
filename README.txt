== Description ==

This WordPress plugin allows users to fetch and display reviews from AllAgents.co.uk directly on their website. With seamless integration, users can showcase authentic reviews from AllAgents.co.uk to build trust and credibility with their audience.


== Installation ==

1. Upload the `allagents-reviews` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings through the 'Settings' or 'Tools' menu.
4. To display reviews fetched from allagents.co.uk/, use the following shortcode:
`allagents-display-reviews`
This shortcode will render the reviews on the front end of your WordPress website.
